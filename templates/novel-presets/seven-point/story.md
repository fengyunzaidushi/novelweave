## 节点间的过渡设计

### Hook → Plot Point 1（第1-2章 → 第X章）

### Plot Point 1 → Pinch Point 1（第X章 → 第X章）

- **适应新现实**：接受改变
- **初步行动**：尝试解决
- **遭遇阻力**：发现困难

### Pinch Point 1 → Midpoint（第X章 → 第X章）

- **压力下成长**：被迫提升
- **寻找突破**：努力改变
- **接近真相**：逐步理解

## 创作备注

### 七点结构核心要求

- [ ] 每个节点必须推进故事
- [ ] 节点间的过渡要自然
- [ ] 保持张力的持续上升
- [ ] 中点必须真正改变游戏
- [ ] 收紧点不能省略
- [ ] 每个点都要提升赌注

### 常见问题避免

- Hook太慢热
- PP1不够重大
- Pinch点太弱
- 中点无力
- PP2准备不足
- 结局草率

### 节奏把控技巧

- 使用短章节加快节奏
- 在节点处切换POV增加冲击
- 用时间压力维持紧张感
- 适时的喘息避免疲劳
